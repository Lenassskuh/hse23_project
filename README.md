## Краткая информация
*Ген*: KDM7A

*Эпигенетическая функция*: Histone modification erase

*Эпигенетическая группа*: H3K27

*Ткани*: костный мозг, толстая кишка, тестикулы, эндометрий и др.

*Домены*: JHD, PHD_KDM7, JmjC, cupin_RmlC-like super family
